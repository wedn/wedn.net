## Clone it

```bash
$ cd path/to/root
$ git clone https://github.com/wedn/wedn.net.git --depth 1 --recursive
$ cd wedn.net
```

## Install dependencies

```bash
$ npm install
# or
$ yarn
```

## Usage

```bash
# build for production with minification
$ npm run build

# watch file changes for development
$ npm run dev
```
